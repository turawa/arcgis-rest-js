UserSession

Class 
const session = new UserSession({
  username: "jsmith",
  password: "123456"
})

Used to manage the authentication of ArcGIS Online and ArcGIS Enterprise users in request. This class also includes several helper methods for authenticating users with OAuth 2.0 in both browser and server applications.
